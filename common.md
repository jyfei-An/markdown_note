# 虚拟机全屏显示

1. 点开菜单栏的 虚拟机---------> 安装VMware Tools
2. 将下载文件拷贝到其他目录中，右键extract here
3. 在终端中输入：sudo perl vmware-install.pl，一直点击回车，完成安装

2. 点击设置
3. 点击”自动填充“
4. 点击“密码”
